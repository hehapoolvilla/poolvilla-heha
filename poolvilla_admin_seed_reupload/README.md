    PoolVilla - Simple prototype

Backend: Node.js + SQLite
Frontend: React + Vite

Seed admin: admin@poolvilla.com / 123456

How to run locally:
1. cd backend
2. npm install
3. npm run seed
4. node src/index.js

Frontend:
1. cd frontend
2. npm install
3. npm run dev

Then open http://localhost:3000
